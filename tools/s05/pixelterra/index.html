PixelTerra 3D Guide - Extruded Edition

1 Borderless Pixel Letters

Unlike traditional blocks, this engine generates only the physical 3D silhouette of each character. Each letter is analyzed and decomposed into active pixels to build individual extruded pillars using the real Micro 5 Google Font.

2 Prevent Overlap with Spacing

Since there are no bounding boxes, if letters are too large or too close, they can look like a single continuous block. To prevent this, use the Letter Spacing slider to add a protective empty boundary around each character.

3 Using Google Maps URLs or Search

Type a well-known place name (e.g., "Teide", "Everest", "Ibiza") or paste a Google Maps link directly from your browser. The AI will interpret the real topography and shape the height of the letters.
